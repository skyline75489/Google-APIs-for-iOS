Google APIs for iOS
===================

Google APIs for iOS 的 spec 仓库，方便墙内使用。

**注意**: 暂时只有 Google/Analytics 的部分

使用方法：

```ruby
pod 'GoogleAnalytics', :podspec => 'https://raw.githubusercontent.com/skyline75489/Google-APIs-for-iOS/master/GoogleAnalytics.podspec.json'
pod 'GoogleInterchangeUtilities', :podspec => 'https://raw.githubusercontent.com/skyline75489/Google-APIs-for-iOS/master/GoogleInterchangeUtilities.podspec.json'
pod 'GoogleNetworkingUtilities', :podspec => 'https://raw.githubusercontent.com/skyline75489/Google-APIs-for-iOS/master/GoogleNetworkingUtilities.podspec.json'
pod 'GoogleSymbolUtilities' ,:podspec => 'https://raw.githubusercontent.com/skyline75489/Google-APIs-for-iOS/master/GoogleSymbolUtilities.podspec.json'
pod 'GoogleUtilities', :podspec => 'https://raw.githubusercontent.com/skyline75489/Google-APIs-for-iOS/master/GoogleUtilities.podspec.json'
pod 'Google/Analytics', :podspec => 'https://raw.githubusercontent.com/skyline75489/Google-APIs-for-iOS/master/Google.podspec.json'
```
